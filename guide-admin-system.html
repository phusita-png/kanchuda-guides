1 ตั้งค่าครั้งแรก
สร้าง/อัปเดตชีตทั้งหมดให้พร้อมใช้
📦 ระบบออเดอร์▸ 🗂️ สร้างชีตทั้งหมด

กดเมนูนี้ ระบบจะสร้างชีตที่จำเป็นทั้งหมด + ใส่หัวตาราง + ช่องติ๊ก/dropdown ให้อัตโนมัติ กดซ้ำได้ปลอดภัย (ไม่ลบข้อมูลเดิม — แค่เติมส่วนที่ขาด)

🔁
เมื่อไหร่ต้องกดซ้ำ? — หลังเพิ่มกลุ่มใหม่ (ให้ dropdown กลุ่มในชีตประกาศอัปเดต) หรือเมื่อระบบมีอัปเดตเพิ่มคอลัมน์
🔐
ครั้งแรกอาจมีหน้าต่างขอ อนุญาต (Authorize) — กด Allow ด้วยบัญชีเจ้าของระบบ (ทำครั้งเดียว)

⚙️ ปรับค่าระบบเองได้ที่ชีต "ตั้งค่า" — แก้เฉพาะ คอลัมน์ B (ค่า) ไม่ต้องแตะโค้ด:

รายการผลต่อระบบ
โฟลเดอร์เก็บไฟล์รายงานวาง Folder ID จาก URL โฟลเดอร์ Drive ที่จะเก็บไฟล์รายงาน WMS
ตอบกลับในกลุ่มเมื่อบันทึกใช่ = บอทตอบการ์ดยืนยัน · ไม่ = เงียบ
เวลาตัดรอบหลัก (ชม.)เช่น 13 = ตัดรอบ 13:00 (Telesales 24 ชม. + ขอบรอบ FB)
เวลาแบ่งเช้า–บ่าย FB (ชม.)เช่น 16 = FB รอบเช้าถึง 16:00
🛡️
ปลอดภัย — เว้นว่าง/กรอกผิด ระบบใช้ค่าเริ่มต้นให้อัตโนมัติ (ไม่พัง)
2 ตั้งค่ากลุ่ม LINE
เชื่อมกลุ่มไลน์เข้าระบบ + กรอกรหัสสำหรับออกรายงาน
  1. เชิญ บอทกานต์ชุดา เข้ากลุ่มไลน์ของพนักงาน
  2. พิมพ์คำว่า groupid ในกลุ่ม → บอทจะบันทึกกลุ่มลงชีต "กลุ่ม" ให้อัตโนมัติ (1 แถว/กลุ่ม)
  3. เปิดชีต "กลุ่ม" แล้วกรอกข้อมูลที่เหลือ ด้วยมือ:
คอลัมน์กรอกอะไร
ชื่อกลุ่มชื่อที่อ่านง่าย เช่น Telesales ก.3 · FB ก.6 (ใช้โชว์ใน dropdown ชีตประกาศด้วย)
รหัสกลุ่มรหัสสั้น ๆ ของกลุ่ม (ออปชัน)
ช่องทางTelesales หรือ FB — ระบบใช้คำนวณเวลาตัดรอบ
orderBaseฐานเลขออเดอร์ เช่น T (Telesales) · A06 (FB)
adminBaseฐานรหัสแอดมินขาย เช่น kan03n-T- · kan03n-06
⚠️
กลุ่มที่ ยังไม่กรอกช่องทาง/orderBase จะถูกข้ามตอนดึงรายงาน (ระบบจะแจ้งว่า "ข้ามเพราะกลุ่มยังไม่ตั้งช่องทาง") → กรอกให้ครบก่อนใช้งาน
3 ตั้งค่าพนักงาน
ผูกชื่อจริง + รหัสคน (ใช้ประกอบเลขออเดอร์/รหัสแอดมินขาย)

มี 2 วิธีให้ชื่อพนักงานเข้าระบบ:

  • อัตโนมัติ — พอพนักงานพิมพ์ออเดอร์ครั้งแรก บอทเก็บ LINE User ID + ชื่อไลน์ลงชีต "พนักงาน" ให้
  • ลงทะเบียนเอง — พนักงานพิมพ์ในกลุ่ม: ลงทะเบียน ชื่อ-นามสกุล รหัส เช่น ลงทะเบียน สมหญิง ใจดี 03
🔑
เปิดชีต "พนักงาน" ตรวจ/เติม รหัสคน (คอลัมน์ E) ให้ครบ เช่น 03 · Z · M — รหัสนี้ผูกกับคน (ไม่ใช่กลุ่ม) คน 1 คนทำได้หลายกลุ่ม
4 ตั้งค่าราคาสินค้า
พจนานุกรมรหัส SKU ให้ระบบจับคู่ได้แม่น

เปิดชีต "ราคาสินค้า" คอลัมน์ A ใส่ รหัส SKU มาตรฐาน ให้ครบทุกตัว เช่น SET-8 · SET-Q3-06

🏷️
ระบบฉลาดพอจะจับคู่ Set 8 / Set8 / SET-8 ให้เป็นรหัสเดียวกัน — แต่ ยิ่งกรอกครบ ยิ่งแปลงรหัสแม่น · ราคาขายจริงระบบ ยึดยอด COD ที่พนักงานพิมพ์ (คอลัมน์ราคาเป็นออปชัน)
5 แจ้งตัดรอบเข้ากลุ่ม ⭐
ให้บอทส่งการ์ดแจ้ง "ตัดรอบแล้ว" เข้ากลุ่มอัตโนมัติ — ตั้งเองได้ที่ชีต "ประกาศ"

ทั้งหมดคุมที่ชีต "ประกาศ" — 1 แถว = 1 รอบแจ้ง จะกี่รอบ/วัน กี่กลุ่มก็ได้ แค่เพิ่มแถว ไม่ต้องแตะโค้ด

คอลัมน์ใส่อะไร
A เปิด✓ ช่องติ๊ก — ติ๊ก = เปิดรอบนี้ · เอาติ๊กออก = ปิด
B กลุ่มเป้าหมายเลือกจาก dropdown — ชื่อกลุ่มที่ลงทะเบียนแล้ว หรือ ทั้งหมด (ทุกกลุ่ม)
C วันที่ส่งdropdown: ทุกวัน / จันทร์-ศุกร์ / เสาร์-อาทิตย์ (หรือพิมพ์เอง จ,พ,ศ)
D เวลาเวลาส่ง เช่น 13:00 · 16:00 (เป๊ะนาที)
E หัวข้อหัวการ์ด เช่น ระบบตัดรอบออเดอร์แล้วค่ะ
F ข้อความเนื้อในการ์ด — ใส่ตัวแปร {วันนี้} {พรุ่งนี้} {เวลา} ได้
G ส่งล่าสุดระบบเขียนเอง (กันส่งซ้ำใน 1 วัน) — ห้ามแก้

ตัวอย่างการตั้งค่า (Telesales วันละครั้ง · FB สองรอบ):

เปิดกลุ่มวันเวลาข้อความ
✓Telesales ก.3 ทุกวัน13:00 ออเดอร์หลัง {เวลา} น. = รอบ {พรุ่งนี้}
✓FB ก.6 ทุกวัน13:00 ตัดรอบเช้าแล้วค่ะ — ออเดอร์หลัง {เวลา} น. = รอบบ่าย
✓FB ก.6 ทุกวัน16:00 ตัดรอบบ่ายแล้วค่ะ — รอบใหม่ {พรุ่งนี้}
🔁
หลายรอบ/วัน = เพิ่มแถว — กลุ่มเดียวกันคนละเวลา ใส่ได้หลายแถว (เช่น FB 13:00 + 16:00 ด้านบน) ระบบกันส่งซ้ำแยกต่อแถว
📅
ตัวแปรอัปเดตเองทุกวัน — การ์ดวันที่ 23 มิ.ย. จะขึ้น {พรุ่งนี้} = 24 มิ.ย. · พอวันถัดไปส่งใหม่ก็เป็น 25 มิ.ย. เอง (การ์ดเก่าในไลน์ไม่เปลี่ยนย้อนหลัง)

เปิดใช้งาน / ทดสอบ — ผ่านเมนู:

📦 ระบบออเดอร์▸ 🔔 แจ้งตัดรอบเข้ากลุ่ม
เมนูย่อยทำอะไร
🧪 ทดสอบส่งตอนนี้ส่งการ์ดทุกแถวที่ติ๊ก ✓ ทันที (ไม่สนวัน/เวลา) — เช็คว่าการ์ดเข้าถูกกลุ่ม + ข้อความถูก
▶️ เปิดแจ้งอัตโนมัติตั้งให้ระบบเช็คทุก 5 นาที แล้วส่งตามตารางในชีตประกาศ (ทำครั้งเดียว · ถ้าเคยเปิดไว้แล้วและมีอัปเดต ให้กดซ้ำ)
⏹️ ปิดแจ้งอัตโนมัติหยุดการส่งอัตโนมัติทั้งหมด
👥
เพิ่มกลุ่มใหม่แล้วไม่เห็นใน dropdown? — ตั้งชื่อกลุ่มในชีต "กลุ่ม" ให้เรียบร้อย แล้วกด 🗂️ สร้างชีตทั้งหมด อีกครั้ง → dropdown จะอัปเดต
⏰
ส่งครั้งเดียวตรงเวลา — ระบบส่งการ์ด 1 ครั้ง/วัน/แถว ในช่วงเวลาที่ตั้ง (ภายใน 15 นาที) เท่านั้น ไม่ส่งซ้ำและไม่ส่งช้าผิดเวลา (กันพนักงานสับสน)
6 งานประจำวัน
ดึงรายงาน · ปิดรอบ · บัญชี · ยอดขาย
งานเมนู / รายละเอียด
🚚 ดึงรายงาน WMSออกไฟล์ Order Template ส่งขนส่ง ตามรอบ → มีคู่มือแยกละเอียด (ดูชุด "ดึงรายงาน WMS")
📦 ปิดรอบเมนู 📑 รายงาน WMS ▸ ปิดรอบ → ย้ายออเดอร์ที่ออกเลขแล้วเข้า Archive (มีถามยืนยันก่อน)
🧾 ส่งออก PEAK (ใบเสร็จรับเงิน)copy ไฟล์ WMS ที่ขนส่งส่งกลับ วางในชีต "WMS ส่งกลับ" ก่อน → เมนู 🧾 PEAK ▸ ออกใบเสร็จรับเงิน → ระบบกรองเฉพาะออเดอร์ "แพ็คเรียบร้อย" (ส่งจริง) แล้วรวมเป็นใบตามคน×วัน → copy ชีต "PEAK" เข้าโปรแกรมบัญชี
📊 Dashboard ยอดขายเมนู 📊 Dashboard ▸ รีเฟรช → ดู KPI + กราฟ (กรอกช่วงวันที่ในชีตได้)
🔗
รายละเอียดการดึงรายงาน เวลาตัดรอบ และช่องสีเหลือง อยู่ในคู่มือ "ดึงรายงานออเดอร์ (WMS)" — กลับไปเลือกที่หน้ารวมคู่มือ
7 ดูแล & แก้ปัญหา
อาการที่พบบ่อยของผู้ดูแลระบบ
อาการสาเหตุ / วิธีแก้
บอทไม่บันทึกออเดอร์ในกลุ่มเช็คว่าบอทยังอยู่ในกลุ่ม + กลุ่มถูกบันทึกในชีต "กลุ่ม" แล้ว (พิมพ์ groupid ซ้ำได้)
การ์ดแจ้งตัดรอบไม่เข้ากลุ่ม(1) ติ๊ก ✓ คอลัมน์ A แล้วยัง · (2) เลือกกลุ่มถูกตัวใน dropdown · (3) บอทยังอยู่ในกลุ่มไหม · ลอง 🧪 ทดสอบส่งตอนนี้
dropdown กลุ่มว่าง/ไม่มีกลุ่มใหม่ตั้งชื่อกลุ่มในชีต "กลุ่ม" ก่อน แล้วกด 🗂️ สร้างชีตทั้งหมด อีกครั้ง
ดึงรายงานแล้วขึ้น "ไม่พบออเดอร์"กลุ่มยังไม่กรอกช่องทาง/orderBase หรือออเดอร์อยู่นอกช่วงเวลารอบ (ดูคู่มือดึงรายงาน)
ขึ้นขออนุญาต (Authorization)ครั้งแรกที่ใช้เมนูที่แตะ Drive/Trigger → กด Allow ด้วยบัญชีเจ้าของระบบ
⚠️
ห้ามแก้/ลบชีต "ออเดอร์" และ "Archive" ด้วยมือ โดยไม่จำเป็น — เป็นข้อมูลต้นทางของรายงานทั้งหมด หากต้องแก้ ให้สำรองไฟล์ก่อน
