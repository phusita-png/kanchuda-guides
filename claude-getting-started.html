01
ทำความรู้จัก

Claude คืออะไร แล้วมี 2 หน้าตายังไง

Claude คือ AI ผู้ช่วยที่คุยโต้ตอบเป็นภาษาไทยได้ ช่วยคิด–เขียน–สรุป–แปล ให้งานเร็วขึ้น · เข้าใช้ได้ 2 ทาง บัญชีเดียวกัน

ทางที่ 1

🌐 claude.ai (บนเว็บ)

  • ✓ เปิดในเบราว์เซอร์ ไม่ต้องติดตั้ง
  • ✓ ใช้ได้ทั้งคอมและมือถือ
  • ✓ เหมาะเริ่มต้น ลองเล่นได้ทันที
ทางที่ 2

💻 Claude Desktop (แอปบนคอม)

  • ✓ เปิดเร็ว อยู่บนคอมตลอด กดเรียกได้ทันที
  • ✓ ลากไฟล์/รูปในเครื่องเข้ามาคุยง่าย
  • ✓ เหมาะเวลาทำงานจริงจังบนคอม
✨
ทั้งสองทางใช้ บัญชี Claude เดียวกัน (แพ็กเกจ Pro $20/เดือน) · เริ่มจากเว็บก่อนก็ได้ แล้วค่อยลงแอปคอมทีหลัง — ประวัติแชทซิงก์ถึงกัน
02
ใช้บนเว็บ · claude.ai

เริ่มคุยกับ Claude บนเว็บ

เปิดเบราว์เซอร์ ล็อกอิน แล้วพิมพ์คุยได้เลย — ง่ายเหมือนแชทกับเพื่อน

เข้า claude.ai แล้วล็อกอิน

เปิดเบราว์เซอร์ → พิมพ์ claude.ai → ล็อกอินด้วยบัญชีที่สมัครไว้ (บัญชี Pro เดียวกับที่ใช้กับระบบ)

พิมพ์สิ่งที่อยากให้ช่วยในช่องล่าง

พิมพ์เป็นภาษาไทยได้เลย เช่น “ช่วยร่างข้อความตอบลูกค้าที่ถามเรื่องการจัดส่ง” แล้วกด Enter

คุยต่อเพื่อปรับให้ถูกใจ

ไม่ต้องได้ครั้งเดียวจบ — พิมพ์ต่อได้ เช่น “ขอสั้นกว่านี้” / “เปลี่ยนเป็นโทนสุภาพ” / “ใส่อีโมจิด้วย” Claude จะปรับให้

แนบไฟล์หรือรูปได้ (ปุ่ม 📎)

ลากไฟล์ Excel รายการสินค้า หรือรูปสลิป/แชตลูกค้า เข้ามาให้ Claude อ่านแล้วช่วยสรุป/จัดตารางได้

🗂️
เคล็ดลับ Projects: ถ้าใช้บ่อย สร้าง “Project” เก็บข้อมูลร้าน (สินค้า/โทนแบรนด์/คำที่ห้ามใช้) ไว้ครั้งเดียว แล้วทุกแชทในโปรเจกต์นั้น Claude จะจำบริบทร้านให้ ไม่ต้องเล่าใหม่ทุกครั้ง
03
แอปบนคอม · Claude Desktop

ติดตั้งแอป Claude ลงคอม

เหมือนเว็บแต่เปิดเร็วกว่า อยู่บนคอมพร้อมเรียกใช้ตลอด — เหมาะเวลานั่งทำงานหน้าคอมจริงจัง

ดาวน์โหลดแอป

เข้า claude.ai/download → กดโหลดเวอร์ชัน Windows → เปิดไฟล์ที่โหลด → กดติดตั้งจนเสร็จ

เปิดแอปแล้วล็อกอิน

เปิดแอป Claude → ล็อกอินด้วยบัญชี Pro เดียวกับบนเว็บ → พร้อมใช้งานทันที (ประวัติแชทเดิมขึ้นมาให้ด้วย)

ปักหมุดไว้ที่แถบงาน (Taskbar)

คลิกขวาที่ไอคอน Claude ที่แถบล่าง → Pin to taskbar → คราวหน้ากดเรียกได้ทันทีไม่ต้องหา

⚡

เปิดไว ใช้ทันใจ

ไม่ต้องเปิดเบราว์เซอร์ หาแท็บ — กดไอคอนเดียวพร้อมพิมพ์

📎

ลากไฟล์ง่าย

ลากไฟล์/รูปจากในเครื่องเข้าหน้าต่างแอปได้ตรง ๆ

🔄

ซิงก์กับเว็บ

คุยบนคอมแล้วเปิดต่อบนมือถือได้ ประวัติเดียวกัน

04
เทคนิคคุยให้ได้งานดี

4 เคล็ดลับ ให้ Claude ตอบตรงใจ

ยิ่งบอกละเอียด ยิ่งได้งานตรง — เหมือนสั่งงานน้องในทีมที่เก่งแต่ต้องบรีฟให้ชัด

🎯

1. บอกบริบทให้ชัด

เล่าว่าคุณคือใคร ขายอะไร ลูกค้าแบบไหน เช่น “ร้านขายครีมบำรุงผิว ลูกค้าเป็นผู้หญิงวัยทำงาน”

🖼️

2. ยกตัวอย่างที่ชอบ

มีแคปชั่น/สไตล์ที่ถูกใจอยู่แล้ว วางให้ดูเป็นตัวอย่าง แล้วบอก “เอาแนวนี้”

📏

3. บอกโทนและความยาว

เช่น “สุภาพเป็นทางการ” / “สนุกวัยรุ่น” / “สั้น 2 บรรทัด” / “มีอีโมจิ”

🔁

4. ขอหลายแบบแล้วเลือก

“ขอ 5 แบบให้เลือก” แล้วค่อยบอก “เอาแบบ 3 แต่ปรับตอนท้าย” — คุยปรับไปเรื่อย ๆ ได้

05
ตัวอย่างงานจริง · กานต์ชุดา

เอาไปใช้ได้เลย — ก็อปแล้วลองเลย

ตัวอย่างคำสั่งที่ใช้ได้จริงกับงานประจำวันของร้าน — กดปุ่มคัดลอกแล้วเอาไปวางใน Claude

✍️ คิดแคปชั่นขายของ
คุณช่วยคิดแคปชั่นขายเซ็ต SET-8 ราคา 799 บาท โทนน่ารักสดใส สั้น ๆ 3 แบบ ใส่อีโมจิ
💬 ตอบแชทลูกค้า
คุณลูกค้าถามว่าครีมนี้ใช้กับผิวแพ้ง่ายได้ไหม ช่วยร่างคำตอบสุภาพ อบอุ่น ไม่การันตีเกินจริง
📊 สรุปยอด/จัดตาราง
คุณนี่คือรายการออเดอร์วันนี้ (วางข้อความ/แนบไฟล์) ช่วยสรุปยอดรวมและนับจำนวนต่อสินค้าเป็นตาราง
🎁 คิดโปรโมชั่น
คุณช่วยคิดไอเดียโปรโมชั่นเดือนหน้าสำหรับร้านครีมบำรุงผิว 5 ไอเดีย พร้อมชื่อแคมเปญ
🌏 แปลภาษา
คุณช่วยแปลรีวิวลูกค้าภาษาอังกฤษนี้เป็นไทย แล้วสรุปว่าลูกค้าพอใจเรื่องอะไร
📣 เขียนประกาศ
คุณช่วยร่างประกาศแจ้งลูกค้าเรื่องหยุดส่งของช่วงสงกรานต์ ให้สุภาพและชัดเจน
06
ข้อควรระวัง

ใช้ให้ปลอดภัยและได้ผลดี

Claude เป็นผู้ช่วยที่เก่ง แต่มี 3 เรื่องที่ควรจำไว้

🔒

อย่าใส่ข้อมูลลับที่ไม่จำเป็น

เลี่ยงการวางเลขบัตรประชาชน รหัสผ่าน เลขบัญชีเต็ม ๆ ของลูกค้า/พนักงาน ลงในแชทถ้าไม่จำเป็นต่องาน

✅

ตรวจก่อนใช้จริงเสมอ

โดยเฉพาะ “ตัวเลข” และ “ข้อเท็จจริง” — Claude ช่วยร่างให้เร็ว แต่คนตัดสินใจสุดท้ายคือเรา อ่านทวนก่อนส่งลูกค้า

🗓️

Claude ไม่รู้ข้อมูลเฉพาะร้านเราเอง

ราคาล่าสุด สต็อก โปรที่กำลังจัด — ต้องบอก/แนบให้ Claude ก่อน ถึงจะช่วยได้ตรง (หรือเก็บไว้ใน Project)

🙌
เริ่มจากลองเล่นวันละนิด — ให้ช่วยงานเล็ก ๆ ก่อน เช่น ร่างแคปชั่น 1 โพสต์ แล้วจะค่อย ๆ จับทางว่าคุยยังไงได้งานดีที่สุด
