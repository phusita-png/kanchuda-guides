02
ใช้บนเว็บ · claude.ai

เริ่มคุยกับ Claude บนเว็บ (ทีละหน้าจอ)

เปิดเบราว์เซอร์ ล็อกอิน แล้วพิมพ์คุยได้เลย — ทำตามทีละขั้น

เปิดเบราว์เซอร์แล้วเข้า claude.ai

เปิด Chrome (หรือ Edge) → คลิกช่องที่อยู่ด้านบนสุด → พิมพ์ claude.ai แล้วกด Enter

💡คือ ช่องที่อยู่ (Address bar) = แถบยาวบนสุดของเบราว์เซอร์ที่พิมพ์ชื่อเว็บ
📸ใส่ภาพ: หน้าแรก claude.ai ก่อนล็อกอินแคปหน้าจอตอนเปิดเว็บมาวางตรงนี้

ล็อกอินด้วยบัญชีที่สมัครไว้

กด Log in → กรอกอีเมล + รหัสผ่าน (หรือปุ่ม “Continue with Google”) → เข้าสู่หน้าแชท

🔑ใช้บัญชี Pro เดียวกับที่ใช้กับระบบ — ถ้าลืมรหัส กด “Forgot password”
📸ใส่ภาพ: หน้าแชทเปล่าหลังล็อกอินให้เห็นช่องพิมพ์ข้อความด้านล่าง

พิมพ์สิ่งที่อยากให้ช่วยในช่องล่าง

คลิกช่องพิมพ์ด้านล่าง → พิมพ์เป็นภาษาไทยได้เลย เช่น “ช่วยสรุปยอดขายจากรายการนี้” → กด Enter (หรือปุ่มลูกศร ➤)

⏎ขึ้นบรรทัดใหม่โดยไม่ส่ง = กด Shift + Enter

แนบไฟล์ Excel หรือรูปสลิป (ปุ่ม 📎)

กดปุ่มคลิปหนีบ 📎 ข้างช่องพิมพ์ → เลือกไฟล์รายการออเดอร์/สลิป → หรือ ลากไฟล์มาวางในช่องแชทตรง ๆ ก็ได้ แล้วพิมพ์บอกว่าให้ทำอะไร

📸ใส่ภาพ: ตอนแนบไฟล์ Excel เข้าไปในแชทให้เห็นไฟล์แนบอยู่เหนือช่องพิมพ์

คุยต่อเพื่อปรับให้ถูกใจ

ไม่ต้องได้ครั้งเดียวจบ — พิมพ์ต่อได้ เช่น “แยกตามสินค้าด้วย” / “ทำเป็นตาราง” / “ใส่ยอด VAT” Claude จะปรับให้ในแชทเดิม

เริ่มแชทใหม่เมื่อเปลี่ยนเรื่อง

ขึ้นงานใหม่ที่ไม่เกี่ยวกับอันเดิม → กด New chat (มุมซ้ายบน) เพื่อให้ Claude เริ่มสด ไม่สับสนกับข้อมูลเก่า

🗂️
เคล็ดลับ Projects: ถ้าใช้บ่อย สร้าง “Project” เก็บข้อมูลร้านไว้ครั้งเดียว (รายการสินค้า+ต้นทุน · เลข VAT · รูปแบบตารางที่สำนักงานบัญชีอยากได้) แล้วทุกแชทในโปรเจกต์นั้น Claude จะจำบริบทให้ ไม่ต้องเล่าใหม่ทุกครั้ง
03
แอปบนคอม · Claude Desktop

ติดตั้งแอป Claude ลงคอม (ทีละหน้าจอ)

เหมือนเว็บแต่เปิดเร็วกว่า อยู่บนคอมพร้อมเรียกตลอด — เหมาะเวลานั่งปิดบัญชีหน้าคอมจริงจัง

ดาวน์โหลดแอป

เข้า claude.ai/download → กดปุ่มโหลดเวอร์ชัน Windows → รอไฟล์โหลดเสร็จ (ดูที่มุมล่างของเบราว์เซอร์ หรือโฟลเดอร์ Downloads)

📸ใส่ภาพ: หน้า claude.ai/download ปุ่ม Windows

เปิดไฟล์แล้วติดตั้ง

ดับเบิลคลิกไฟล์ที่โหลดมา → ถ้า Windows ถามยืนยัน กด Yes / Run → รอติดตั้งจนเสร็จ (ไม่กี่วินาที)

เปิดแอปแล้วล็อกอิน

เปิดแอป Claude ที่เพิ่งลง → ล็อกอินด้วยบัญชี Pro เดียวกับบนเว็บ → พร้อมใช้งานทันที (ประวัติแชทเดิมขึ้นมาให้ด้วย)

📸ใส่ภาพ: หน้าแอป Claude Desktop หลังเปิด

ปักหมุดไว้ที่แถบงาน (Taskbar)

คลิกขวาที่ไอคอน Claude ที่แถบล่างของจอ → เลือก Pin to taskbar → คราวหน้ากดเรียกได้ทันทีไม่ต้องหา

📌Taskbar = แถบล่างสุดของจอ Windows ที่มีไอคอนโปรแกรมเรียงอยู่
⚡

เปิดไว ใช้ทันใจ

ไม่ต้องเปิดเบราว์เซอร์ หาแท็บ — กดไอคอนเดียวพร้อมพิมพ์

📎

ลากไฟล์ง่าย

ลากไฟล์ Excel/รูปสลิปจากในเครื่องเข้าหน้าต่างแอปได้ตรง ๆ

🔄

4 เคล็ดลับ ให้ Claude ตอบตรงใจ

ยิ่งบอกละเอียด ยิ่งได้งานตรง — เหมือนสั่งงานผู้ช่วยที่เก่งแต่ต้องบรีฟให้ชัด

🎯

1. บอกบริบทให้ชัด

เล่าว่าร้านขายอะไร ต้องการรายงานแบบไหน เช่น “ร้านขายครีม กาแฟ ชาเขียวแบบซอง อยากได้สรุปยอดแยกเงินสด/โอน ส่งสำนักงานบัญชี”

📄

2. ให้ข้อมูล/ตัวอย่างที่มี

วางรายการออเดอร์ หรือแนบไฟล์ Excel เข้าไป — ยิ่ง Claude เห็นข้อมูลจริง ยิ่งสรุปได้แม่น

📏

3. บอกรูปแบบผลลัพธ์

เช่น “ทำเป็นตาราง” / “แยกคอลัมน์ วันที่–สินค้า–ยอด–วิธีชำระ” / “สรุปเป็นข้อ ๆ” / “บอกยอดรวมท้ายตาราง”

🔁

4. ตรวจแล้วขอปรับ

อ่านผลแล้วบอกแก้ได้ เช่น “ยอด VAT คิดจากยอดก่อนภาษีนะ” / “แยกกาแฟกับชาเขียวคนละแถว” — คุยปรับไปเรื่อย ๆ ได้

05
ตัวอย่างงานบัญชีจริง · กานต์ชุดา

เอาไปใช้ได้เลย — ก็อปแล้วลองเลย

ตัวอย่างคำสั่งงานบัญชี/การเงินของร้าน กดปุ่มคัดลอกแล้ววางใน Claude พร้อมแนบข้อมูล/ไฟล์ของจริง

📊 งานบัญชี & การเงิน

📊 สรุปยอดขายรายวัน
คุณนี่รายการออเดอร์วันนี้ (วาง/แนบไฟล์) ช่วยสรุปยอดขายรวม จำนวนออเดอร์ และแยกตามสินค้า (ครีม/กาแฟ/ชาเขียว) เป็นตาราง
💰 กระทบยอดเงินเข้า
คุณช่วยเทียบยอดโอนเข้าบัญชี (แนบสเตทเมนต์) กับรายการออเดอร์โอน ว่ายอดตรงกันไหม มีออเดอร์ไหนที่ยังไม่เข้าเงิน
🧾 คำนวณ VAT
คุณยอดขายรวม (รวม VAT แล้ว) คือ 24,580 บาท ช่วยแยกยอดก่อน VAT และ VAT 7% ให้เห็นชัด ๆ
📁 รายงานส่งสำนักงานบัญชี
คุณช่วยจัดสรุปยอดขายเดือนนี้เป็นตารางสำหรับส่งสำนักงานบัญชี แยกเงินสด(COD)/โอน และแสดงยอดก่อน VAT + VAT + ยอดสุทธิ
📦 แยก COD ที่ต้องตามเก็บ
คุณจากรายการนี้ ช่วยแยกออเดอร์ COD ที่ขนส่งยังไม่โอนเงินคืน พร้อมยอดรวมที่ยังค้างรับ
🔢 คำนวณกำไรต่อสินค้า
คุณเซ็ต SET-8 ต้นทุน 400 บาท ขาย 799 บาท ช่วยคำนวณกำไรต่อชิ้นและ % กำไร แล้วถ้าขายได้ 30 ชิ้นได้กำไรเท่าไร
📉 สรุปรายรับ–รายจ่าย
คุณนี่รายการรายรับรายจ่ายเดือนนี้ (แนบไฟล์) ช่วยสรุปกำไรขั้นต้น และจัดหมวดค่าใช้จ่ายที่สูงสุด 3 อันดับ
📅 เทียบยอดข้ามเดือน
คุณช่วยเทียบยอดขายเดือนนี้กับเดือนที่แล้ว บอกว่าเพิ่ม/ลดกี่ % และสินค้าตัวไหนขายดีขึ้นชัดสุด

✍️ งานขาย & ลูกค้า (โบนัส)

✍️ คิดแคปชั่นขายของ
คุณช่วยคิดแคปชั่นขายเซ็ตครีม SET-8 ราคา 799 บาท โทนน่ารักสดใส สั้น ๆ 3 แบบ ใส่อีโมจิ
💬 ตอบแชทลูกค้า
คุณลูกค้าถามว่ากาแฟซองนี้หวานมากไหม เหมาะกับคนคุมน้ำตาลหรือเปล่า ช่วยร่างคำตอบสุภาพ ชัดเจน ไม่การันตีเกินจริง
✅
ตรวจตัวเลขทุกครั้งก่อนใช้จริง — Claude ช่วยคิดเลขได้เร็วและดี แต่กับ “ตัวเลขเงิน” ที่ส่งสำนักงานบัญชี/ยื่นภาษี ควรอ่านทวนหรือกดเครื่องคิดเลขเช็คจุดสำคัญเสมอ
06
คำถามที่พบบ่อย

FAQ & แก้ปัญหา

คำถามที่มักเจอตอนเริ่มใช้ — คลิกที่คำถามเพื่อดูคำตอบ

พิมพ์ภาษาไทยได้ไหม ต้องเป็นภาษาอังกฤษหรือเปล่า?
พิมพ์ไทยได้เลยค่ะ Claude เข้าใจและตอบไทยได้ดี ไม่ต้องแปลเป็นอังกฤษ
Claude คิดเลขบัญชีถูกต้องไหม?
ช่วยคิดได้ดีและเร็ว โดยเฉพาะสรุป/จัดตาราง แต่ กับตัวเลขเงินที่สำคัญ ควรตรวจทานเสมอ — ให้ Claude ช่วยร่าง แล้วเราเป็นคนเช็คจุดตัดสินใจสุดท้าย
แนบไฟล์ Excel หรือรูปสลิปได้ไหม?
ได้ค่ะ กดปุ่มคลิป 📎 หรือลากไฟล์มาวางในช่องแชท แล้วพิมพ์บอกว่าให้ทำอะไร เช่น “สรุปยอดจากไฟล์นี้เป็นตาราง”
ข้อมูลที่พิมพ์เข้าไปปลอดภัยไหม?
การสนทนาเป็นส่วนตัวของบัญชีเรา แต่ควร เลี่ยงใส่ข้อมูลลับที่ไม่จำเป็น เช่น เลขบัตรประชาชน/รหัสผ่าน/เลขบัญชีเต็มของลูกค้า (ดูหัวข้อ “ข้อควรระวัง”)
ต้องจ่ายเพิ่มไหม นอกจากค่า Pro?
ไม่ต้องค่ะ แพ็กเกจ Pro $20/เดือน ใช้ได้ทั้ง claude.ai (เว็บ) และ Claude Desktop (แอปคอม) ด้วยบัญชีเดียว
แชทเก่าหายไหม กลับไปดูได้ไหม?
ไม่หายค่ะ แชทเก่าเก็บอยู่ในแถบประวัติด้านซ้าย คลิกเปิดดูย้อนหลังได้ — ตั้งชื่อแชทให้จำง่ายก็ได้
Claude ตอบผิด/ไม่ตรงที่ต้องการ ทำยังไง?
พิมพ์บอกแก้ได้เลย เช่น “ไม่ใช่แบบนี้ ขอแยกคอลัมน์วิธีชำระด้วย” — ยิ่งบอกชัดว่าอยากได้อะไร ยิ่งตรง · หรือกด New chat เริ่มใหม่พร้อมบรีฟที่ละเอียดกว่าเดิม
ให้ช่วยงานทั้งบัญชีและการตลาดในแชทเดียวได้ไหม?
ได้ แต่แนะนำ แยกแชท/แยก Project ตามเรื่อง จะเป็นระเบียบและ Claude ไม่สับสนบริบท เช่น แชทหนึ่งไว้ปิดบัญชี อีกแชทไว้คิดแคปชั่น
