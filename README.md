# คู่มือระบบรับออเดอร์ผ่าน LINE — กานต์ชุดา

เว็บคู่มือการใช้งานระบบรับออเดอร์ผ่าน LINE ของบริษัทกานต์ชุดา (เผยแพร่ผ่าน GitHub Pages)

## 🔗 ลิงก์ใช้งาน
- **หน้ารวมคู่มือ:** https://phusita-png.github.io/kanchuda-guides/
- 📱 คู่มือพนักงานออเดอร์: [`guide-staff-order.html`](guide-staff-order.html)
- 📑 คู่มือคนดึงรายงาน WMS: [`guide-puller-wms.html`](guide-puller-wms.html)

## 📄 เนื้อหา
| ไฟล์ | สำหรับ | สรุป |
|------|--------|------|
| `index.html` | ทุกคน | หน้ารวม เลือกคู่มือตามหน้าที่ |
| `guide-staff-order.html` | พนักงานขาย | วิธีพิมพ์ออเดอร์เข้ากลุ่มไลน์ (ลงทะเบียน + รูปแบบ + ตัวอย่าง) |
| `guide-puller-wms.html` | แอดมิน | วิธีกดดึงรายงาน WMS ตามรอบ + ตรวจราคา + แก้ปัญหา |

> คู่มือเป็น HTML ล้วน เปิดได้ทั้งมือถือและคอมพิวเตอร์ ไม่มีข้อมูลลับของระบบ
